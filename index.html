Explain br tags

Br stands Break tags.It is an empty tag. It is used to create a line break in a text. It is generally used in poems and addresses where the divsion of line is necessary.

Expalin how many types of lists?

There are three types of lists-

  • Unordered Lists
  • Ordered Lists
  • Description lists

1.Unordered Lists- are indented lists with a specia bullet symbol in front of each item.

EXAMPLE-

  • MOUSE
  • KEYBOARD
  • CPU
  • HARDDISK
  • CD
  • PRINTER

2.Ordered Lists-are indented lists that have numbers or letters in front of each item.

  1. KEYBOARD
  2. MOUSE
  3. CPU

3.Description lists -are indented lsts without any bullet symbol or any number in front of each item

What are semantic tags?

A semantic element clearly describes its meaning to both the browser and the developer. Examples of non-semantic elements: Div and span - Tells nothing about its content. Examples of semantic elements: Form, Table, and Article - Clearly defines its content.
